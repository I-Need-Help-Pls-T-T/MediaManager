# Activity Diagrams for Media Manager

This section presents the key Activity Diagrams, modeling the main user scenarios and business logic of the system.

---

## UC1: User Registration

The diagram models the process of registering a new user, including the cycle for checking and correcting input errors.

![UC1 Register User](docs\sсhema\activity_diagrams\register.png)

## UC2: Manage Personal Collection

The diagram describes the interaction of a regular user with the catalog and their personal collection: searching, adding media, changing status, rating, and commenting.

![UC2 Personal Collection](docs\sсhema\activity_diagrams\collection.png)

## UC3: Admin Catalog Management

The diagram models the content management process, which is available only to the administrator: adding, editing, and deleting media in the public catalog.


![UC3 Admin Catalog Management](docs\sсhema\activity_diagrams\admin.png)
